# 🚗 MiniGTA 2D — Городской Хаос

**2D top-down экшен в стиле классической GTA 2**

Интерактивная игра с открытым миром, автомобилями, полицией, оружием и динамичными миссиями.

## ✨ Особенности

- **Открытый большой город** (4096×5120 пикселей)
- **Управление автомобилями** с реалистичной физикой и разными характеристиками
- **Система розыска** (Wanted Level) — чем выше уровень, тем агрессивнее полиция
- **Оружие** — пистолет и автомат с перезарядкой
- **Миссии**:
  - Угон дорогой машины
  - Rampage (убить N человек за время)
  - Гонка на время с чекпоинтами
- **Полицейские** с ИИ — преследуют, выходят из машин и стреляют
- **Пешеходы** и случайные события
- **Интерьеры зданий** — можно заходить в дома
- **Система здоровья**, денег и уровня розыска

## 🎮 Как играть

- **WASD** — движение персонажа
- **E** — сесть/выйти из машины
- **ЛКМ** — стрелять (если есть оружие)
- **1 / 2 / 3** — выбор оружия
- **I** — начать миссию (рядом с жёлтым маркером)
- **Правая кнопка мыши** — вращение камеры (в 3D-проектах, здесь не используется)

## 🛠 Технологии

- **Язык**: C#
- **Фреймворк**: Windows Forms + GDI+
- **Физика и коллизии**: Пиксельные маски (`createMask()`)
- **Pathfinding**: A* с поддержкой полос движения (`LaneAStar`)
- **ИИ**: Простой ИИ для пешеходов и полицейских

## 🗂 Структура проекта

- `Map.cs` — основной класс карты и игровой логики
- `Car.cs` — базовый класс автомобиля + `PoliceCar`
- `Player.cs` — управление главным героем
- `Humon.cs` / `PoliceUnit.cs` — пешеходы и полицейские
- `House.cs` — здания с интерьерами
- `Mission.cs` — система миссий
- `LaneAStar.cs` — алгоритм поиска пути
- `WayMask.cs` — маска дорог и спавн-поинты

## 🎯 Цели проекта

- Реализация полноценного небольшого открытого мира
- Создание увлекательного геймплея с несколькими механиками
- Практика работы с графикой GDI+, коллизиями и ИИ
